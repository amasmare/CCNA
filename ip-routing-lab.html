إعداد المسارات الثابتة (Static Routes)

اختر الأمر الصحيح لكل سيناريو توجيه ثابت.

📐 صيغة الأمر

! المسار الثابت الأساسي:
ip route [الشبكة الوجهة] [قناع الشبكة] [Next-Hop أو المنفذ الخارج]
! أمثلة:
ip route 192.168.2.0 255.255.255.0 10.0.0.2
ip route 0.0.0.0 0.0.0.0 10.0.0.1 ← مسار افتراضي

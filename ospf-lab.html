مفاهيم OSPF الأساسية

OSPF بروتوكول توجيه Link-State يبني خريطة كاملة للشبكة. طابق المصطلحات بتعريفاتها.

📚 مصطلحات OSPF — مطابقة

📡 مراحل إنشاء علاقة الجوار (Adjacency)

1. Down
لا يوجد Hello مستقبَل بعد
2. Init
تم استقبال Hello لكن Router ID غير موجود
3. 2-Way
يرى كل منهما نفسه في Hello الآخر
4. ExStart
تحديد Master/Slave لتبادل LSDB
5. Exchange
تبادل ملخصات LSA (DBD Packets)
6. Loading
طلب LSAs الناقصة وإرسالها
7. Full ✓
LSDB متزامنة تماماً — جاهز للتوجيه

حساب تكلفة OSPF (Cost)

تكلفة OSPF = المرجع (100 Mbps) ÷ عرض النطاق. الراوتر يختار المسار ذا أقل تكلفة مجموعة.

إعداد OSPF على Cisco IOS

راجع الأوامر الصحيحة ثم أجب على الأسئلة.

🖥️ تسلسل إعداد OSPFv2

! تفعيل OSPF
R1(config)# router ospf 1 ← Process ID (محلي فقط، لا يجب تطابقه)
! الإعلان عن الشبكات
R1(config-router)# network 192.168.1.0 0.0.0.255 area 0
R1(config-router)# network 10.0.0.0 0.0.0.3 area 0
! Router ID يدوي (اختياري)
R1(config-router)# router-id 1.1.1.1
! التحقق
R1# show ip ospf neighbor ← عرض الجيران
R1# show ip ospf database ← عرض LSDB
R1# show ip route ospf ← مسارات OSPF في جدول التوجيه
💡 Wildcard Mask هو عكس قناع الشبكة: 255.255.255.0 → Wildcard = 0.0.0.255

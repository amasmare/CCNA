📋 الفرق بين Trunk وAccess

منفذ Access
• ينتمي لـ VLAN واحدة فقط
• يتصل بالأجهزة النهائية (PC، طابعة، هاتف)
• البيانات تمر بدون علامة 802.1Q
• الأمر: switchport mode access
منفذ Trunk
• يحمل بيانات عدة VLANs
• يتصل بين سويتشات أو سويتش وراوتر
• البيانات تُوسَم بـ 802.1Q Tag
• الأمر: switchport mode trunk

أوامر إعداد VLANs في Cisco IOS

راجع الأوامر الصحيحة لإنشاء VLANs وتعيين المنافذ ثم أكمل الفراغات.

🖥️ تسلسل إعداد VLAN كاملاً

SW1# configure terminal
! إنشاء VLAN 10 وتسميتها
SW1(config)# vlan 10
SW1(config-vlan)# name Accounting
SW1(config-vlan)# exit
! تعيين منفذ Access للـ VLAN 10
SW1(config)# interface FastEthernet0/1
SW1(config-if)# switchport mode access
SW1(config-if)# switchport access vlan 10
SW1(config-if)# exit
! إعداد منفذ Trunk
SW1(config)# interface GigabitEthernet0/1
SW1(config-if)# switchport mode trunk
SW1(config-if)# switchport trunk allowed vlan 10,20,30
! التحقق
SW1# show vlan brief
SW1# show interfaces trunk

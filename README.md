## 📖 About the Project

This project is a set of **10 self-contained HTML lab files** designed to help students study and practice for the **Cisco CCNA 200-301** certification exam. All labs are:

- 🌙 **Dark-themed** with a unique accent color per lab
- 🌍 **Fully in Arabic (RTL)** — built for Arabic-speaking learners
- 📦 **No internet required** — open any file directly in a browser
- 💾 **Score persistence** — quiz results are saved via `localStorage`
- 📱 **Responsive** — works on desktop and mobile

Each lab includes **interactive exercises** (matching, dropdowns, scenario questions) and a **10-question quiz** with instant feedback.

## 🏗️ Lab Structure

Every lab follows the same layout:

```
Topbar      →  Navigation links to all 10 labs + home
Lab Header  →  Gradient banner with lab title, badge & chapter tags
Nav Bar     →  Back link · Quiz score chip · Reset & Save buttons
Tabs        →  3 exercise tabs + 1 quiz tab
Footer      →  Author credit
```

### Tab Types Used Across Labs

| Tab Type | Description |
|----------|-------------|
| **Matching** | Click a term then click its definition to match pairs |
| **Dropdown** | Select the correct answer from a menu |
| **Radio / Options** | Choose the correct scenario answer |
| **Text Input** | Type subnet addresses or network values |
| **Scenario** | Read a topology then answer multi-part questions |
| **CLI Reference** | Read `show` command output and answer questions |
| **Interactive Calculator** | Subnet calculator (network address, broadcast, hosts) |
| **Quiz** | 10-question multiple choice with instant feedback |

---

## 🚀 How to Use

1. **Clone or download** the repository
2. **Open `index.html`** in any browser to see all labs
3. **Click any lab card** to open that lab
4. Work through each tab — exercises give immediate right/wrong feedback
5. Complete the **Quiz tab** to save your score
6. Use **حفظ** to save your score or **إعادة ضبط** to reset and start over

> No installation, no server, no internet needed.
